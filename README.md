# Node-Blog

    use ( Node + Express + Vue ) make a The front-end separation self-blog.

## How to use ?

client内是完整前端项目，server是完整后端项目
```bash
# 打开终端，启动前端开发环境
cd client & npm  install
npm run dev:client

# 再次打开终端，启动后端环境
cd server & npm  install
npm run dev:server

# 部署
# 前端打包
npm run build:client
# 后端无需打包

```

### Enviroment

1. **node  ==  v6.12.0**

2. **npm   == v5.7.1**

3. **vue   == v2.5.2**

3. **webpack  ==  v3.6.0**
